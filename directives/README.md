# Directives

This directory contains Standard Operating Procedures (SOPs) and high-level instructions (Layer 1) for the 3-layer architecture.

## Organization
- Each file should define a clear goal, required inputs, and the specific execution scripts to run.
- Update these documents as you learn about API constraints or better approaches.

## Operating Principles
1. **Goal-Oriented**: Focus on what needs to be achieved.
2. **Layer Separation**: Directives define "what", Execution scripts define "how".
3. **Living Documents**: Improve continuously with each run.
